# ArsenDiscord
---
Веб-проект в стиле Discord с логикой покупки подписки.  
На странице checkout пользователь заполняет форму, после нажатия **«Купить подписку»**:

1. данные отправляются на сервер;
2. создается `.txt` файл с информацией о покупке;
3. файл сохраняется на сервере в `cards/`;
4. этот же `.txt` автоматически скачивается в браузере.

## Функционал

- Discord-style лендинг с тарифами
- Переход на checkout с выбранным планом (`?plan=...`)
- Обработка формы подписки
- Генерация `.txt` файла покупки
- Автоматическое скачивание файла на клиенте
- Сохранение копии файла на сервере

## Технологии

- HTML / CSS / JavaScript
- Node.js
- Express

## Структура

- `index.html` — главная страница с тарифами
- `checkout.html` — оформление подписки
- `server.js` — API и сохранение файлов
- `cards/` — созданные `.txt` файлы
- `package.json` — зависимости и запуск

## Запуск проекта

bash
npm install
npm start

По умолчанию сервер доступен на: http://localhost:3001
